# Gas Leak Identification and Alert System

## Aim
To design and develop a gas leak detection system using ESP8266 that alerts users when gas leakage is detected.

## Problem Statement
Gas leaks in homes and industries can cause dangerous accidents like fire and explosions. Early detection helps prevent such hazards.

## Scope
- Detect gas leakage using MQ2 sensor
- Send alerts using buzzer and LED
- Real time monitoring using ESP8266
- Improve safety in homes and industries

## Components Used
- ESP8266 NodeMCU
- MQ2 Gas Sensor
- Buzzer
- LED
- Resistors
- Breadboard
- Jumper wires

## Software Used
- Arduino IDE
- Tinkercad / Fritzing
- GitHub

## Working
The MQ2 sensor detects gas concentration. If the gas level exceeds the threshold value, the ESP8266 triggers a buzzer and LED alert.
